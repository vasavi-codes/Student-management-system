# 🎓 Student Management System

## 📌 Description
A C++-based Student Management System designed to manage student records efficiently. This project demonstrates core programming concepts like object-oriented programming and file handling.

## 🚀 Features
- Add student details
- Display student information

## 🛠️ Technologies Used
- C++

## ▶️ How to Run
1. Compile using any C++ compiler
2. Run the program
3. Enter student details when prompted

## 📌 Future Improvements
- Add multiple student support
- File handling for data storage
- Menu-driven system
